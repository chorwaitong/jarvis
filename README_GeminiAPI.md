## Setting up Gemini API
### Steps:
1. Sign up or log in to the [Gemini AI Studio](https://aistudio.google.com/app/apikey).
2. Generate an API key from the dashboard.
3. Save the API key as an environment variable with name `GOOGLE_API_KEY`: In Windows, search "Environment" and there will be suggestion leading to "Edit the system environment variable", navigate to "Environment Variables" and create a new system variable accordingly.
